Code Igniter + Rapyd Library Installation Instructions

  1. Unzip the codeigniter_rapyd package.
  2. Upload the content of main folder to your server. The index.php file will be at your root.
  3. Open the application/config/config.php file with a text editor and set your base URL.
  4. If you intend to use a database, open the application/config/database.php file with a text editor and set your database settings.
  5. In order to run Rapyd Samples controllers, you need to build some tables in your database. See the index page of samples (http://yourhost/index.php/rapyd/samples) and dump the proposed query to yuor database, then you can run all samples

Rapyd Library manual Installation Instructions in your CI 1.5.* Installation

  1. Unzip the codeigniter_rapyd package.
  2. Copy /application (entire folder) to your Code Igniter application folder.
  3. In order to run Rapyd Samples controllers, you need to build some tables in your database. See the index page of samples (http://yourhost/index.php/rapyd/samples) and dump the proposed query to yuor database, then you can run all samples

Note 1: Rapyd extend some of CI libraries (using MY_Libname subclassing): Pagination and Validation.

Note 2: Rapyd add 5 language translations to CI, trash /application/language if you don't need multilanguage support

That's it!
